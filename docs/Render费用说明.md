## 💰 什么时候会收费？

### 可能收费的情况

1. **运行时间超过 750 小时/月**
   - 多个服务叠加
   - 超过一个实例运行

2. **数据库存储超过 256MB**
   - 存储大量数据
   - 存储文件（图片、视频等）

3. **带宽超过 100GB/月**
   - 大量文件下载
   - 视频流媒体

4. **选择了付费计划**
   - 创建时选择了 "Standard" 或 "Pro"
   - 手动升级了配置

## 🎯 如何避免收费？

### 1. 确保选择免费计划

创建服务时：
- ✅ 选择 **"Free"** 计划
- ❌ 不要选择 "Standard" 或 "Pro"

### 2. 检查当前配置

在 Render 控制台：
1. 进入你的服务详情页
2. 查看 **"Settings"** → **"Plan"**
3. 确认显示为 **"Free"**

### 3. 监控资源使用

定期检查：
- **运行时间**：确保不超 750 小时/月
- **数据库存储**：保持在 256MB 以下
- **带宽使用**：控制在 100GB/月以内

### 4. 优化存储

如果数据库存储接近 256MB：
- 清理不必要的数据
- 将大文件（图片、PDF）存储到对象存储
- 定期备份数据并清空历史数据
