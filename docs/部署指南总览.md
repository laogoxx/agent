## 🚀 快速开始

### 使用 Render 部署（推荐）

```bash
# 1. 准备项目
bash scripts/prepare_render.sh

# 2. 提交代码
git add .
git commit -m "准备 Render 部署"
git push

# 3. 在 Render 上创建项目
# 访问 https://dashboard.render.com
# 点击 New + → New Blueprint
# 选择你的 GitHub 仓库
# 点击 Apply
```

详见：[Render 快速开始](./Render快速开始.md)

---

### 使用云服务器部署

```bash
# 1. 连接服务器
ssh root@你的服务器IP

# 2. 上传项目文件
scp -r /workspace/projects/* root@你的服务器IP:/opt/opc-agent/

# 3. 运行部署脚本
cd /opt/opc-agent
chmod +x scripts/deploy.sh
./scripts/deploy.sh
```

详见：[快速部署指南](./快速部署指南.md)

## 📊 部署方案详细对比

### Render

**配置**：
- CPU：0.1 核（免费）
- 内存：512MB（免费）
- 数据库：256MB（免费）
- 带宽：100GB/月（免费）

**价格**：
- 免费：$0/月（小型应用）
- 标准：约 $5-10/月（中型应用）
- 专业：约 $20-50/月（大型应用）

**优势**：
- 零配置部署
- 自动扩展
- 免费 SSL
- 内置数据库
- 自动备份

**劣势**：
- 资源限制较多
- 无法完全控制
- 需要绑定 GitHub

---

### 云服务器（阿里云/腾讯云）

**配置**：
- CPU：1-2 核
- 内存：2-4GB
- 带宽：1-3Mbps

**价格**：
- 最低：约 30-50元/月
- 推荐：约 60-100元/月
- 高配：约 200-500元/月

**优势**：
- 完全控制
- 资源充足
- 灵活配置
- 可部署多个项目
- 数据安全

**劣势**：
- 需要手动配置
- 需要维护
- 需要购买数据库（或使用 PostgreSQL）

## ❓ 常见问题

### Q: 我应该选择哪种部署方式？

**A**:
- 如果你是新手，想要快速部署：选择 **Render**
- 如果你是开发者，需要完全控制：选择 **云服务器**
- 如果你只是测试和学习：选择 **本地运行**

### Q: Render 免费层够用吗？

**A**:
对于小型应用（日访问量 < 1000），免费层完全够用。
如果访问量较大，建议升级到付费计划。

### Q: 云服务器需要多少钱？

**A**:
最低配置（1核2G）：约 30-50元/月
推荐配置（2核4G）：约 60-100元/月
高配配置（4核8G）：约 200-300元/月

### Q: 可以切换部署方式吗？

**A**:
可以。代码是通用的，可以从一种部署方式切换到另一种方式。

### Q: 如何备份数据？

**A**:
- Render：自动备份（每24小时）
- 云服务器：手动备份或使用云备份服务
